[前端筆記] CSS Image Sprite

2016-04-07

當網站有許多圖片特效(例如游標移到圖片上或點擊)，如果採用一個動作搭配CSS設定background image，會產生不必要的http request，增加載入的時間。
我們可以事先將圖片接在一起，在特定動作發生時，使用background-position改變圖片的顯示
這樣就能減少http request的次數

這二張皆為64x64 pixels，以下為hover的效果範例
